# legal-dep
Клиентская часть веб приложения "Юридический портал". Используемые технологии: vue.js, typescript, vue-material-design.

## Запуск проекта:
Нужно выполнить две комманды:

### Загрузка необходимых библиотек
```
npm install
```

### Локальный запуск в dev режиме
```
npm run serve
```
